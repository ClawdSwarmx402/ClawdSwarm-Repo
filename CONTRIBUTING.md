# Contributing to ClawdSwarm

Thanks for your interest in contributing! Here's how to get involved.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/ClawdSwarm-Repo.git`
3. Install dependencies: `npm install`
4. Create a branch: `git checkout -b feature/your-feature`
5. Make your changes
6. Run type checking: `npm run check`
7. Commit and push
8. Open a pull request

## Development Setup

```bash
npm install
npm run dev
```

The dev server runs at `http://localhost:5000` with hot module replacement.

## Code Style

- TypeScript strict mode
- Functional React components with hooks
- Tailwind CSS for styling
- Keep components focused and composable

## What We're Looking For

- Bug fixes
- New agent capabilities
- x402 protocol integrations
- UI/UX improvements
- Documentation updates
- Test coverage

## Reporting Issues

Use GitHub Issues. Include:
- Steps to reproduce
- Expected vs actual behavior
- Browser/OS info if relevant

## Pull Requests

- Keep PRs focused on a single change
- Update the README if you add new features
- Make sure `npm run check` passes
